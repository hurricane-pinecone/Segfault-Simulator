# Runtime systems

The runtime ships built-in systems for common mechanics. Register them on a
scene and they run each frame during the update and render phases.

- **[Camera system](./camera-system/index.md)** — smooth entity-follow camera
  with configurable lag and world-space offset.

For how systems fit into the scene lifecycle, see
[Game & scenes](../game-and-scenes/index.md). For the render systems
(`SpriteRenderSystem`, `FlatRenderSystem`, `IsometricRenderSystem`), see
[Rendering](../rendering/index.md).
